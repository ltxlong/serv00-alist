# serv00-alist
在serv00上一键部署alist，并且实现访问即拉起和保持更新最新版


基于 [jinnan11/serv00-api](https://github.com/jinnan11/serv00-api) 修改

区别：本项目进行精简，只用一个端口（serv00限制每个账号只有3个端口，省着用）
